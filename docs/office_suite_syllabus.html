Track: Beginner Advanced

Beginner Level (12 hours)

Module Topics & Hands-on
1. Getting Started Interface, creating/saving documents, templates, basic navigation, file formats (.docx, .pdf).
Hands-on: create a basic resume.
2. Text Formatting Font styles, paragraph alignment, bullets & numbering, line spacing.
Hands-on: format a one-page letter.
3. Page Layout Margins, headers/footers, page orientation, columns.
Hands-on: create a two-column newsletter page.
4. Working with Objects Pictures, shapes, WordArt, text boxes, wrapping.
Hands-on: design a flyer with images and captions.
5. Tables & Lists Insert/edit tables, table styles, converting text to table.
Hands-on: create attendance table and style it.
6. Proofing & Review Spellcheck, comments, track changes, simple collaboration.
Hands-on: review and accept/reject changes in a shared doc.

Advanced Level (18 hours)

Module Topics & Hands-on
1. Styles & Templates Creating and applying styles, building templates, theme management.
Hands-on: build an invoice template with styles.
2. Advanced Table Techniques Nested tables, table formulas, sorting/filtering within tables.
Hands-on: create an expense tracker using tables.
3. Sections & Long Documents Section breaks, page numbering, table of contents, index.
Hands-on: prepare a short report with TOC and different headers per section.
4. Mail Merge Data sources (Excel), merge fields, letters/envelopes/labels.
Hands-on: mail-merge a batch of invitation letters from Excel data.
5. Macros & Automation Record simple macros, assign buttons, basics of security.
Hands-on: record macro to format headings and apply styles.
6. Document Protection Restrict editing, password protection, accessibility checker, PDF export.
Hands-on: secure a policy document and export to accessible PDF.
